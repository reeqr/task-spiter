## MODIFIED Requirements

### Requirement: 出题角度结果展示
系统 MUST 在独立弹窗中展示术语或概念的命题角度分析结果，并在追问阶段支持文本流式输出。

#### Scenario: 查询成功后展示
- **WHEN** AI 返回出题角度结果
- **THEN** 系统 MUST 打开弹窗并展示对应术语名称和结果文本

#### Scenario: 查询失败提示
- **WHEN** AI 查询出题角度失败
- **THEN** 系统 MUST 提示失败信息并恢复按钮可点击状态

#### Scenario: 结果隔离
- **WHEN** 用户连续查询不同术语的出题角度
- **THEN** 每次弹窗内容 MUST 对应当前术语，不得复用上一次术语结果

#### Scenario: 出题角度追问流式追加
- **WHEN** 用户在出题角度弹窗提交追问且服务端返回流式文本
- **THEN** 系统 MUST 按到达顺序持续追加文本到当前追问回复中
