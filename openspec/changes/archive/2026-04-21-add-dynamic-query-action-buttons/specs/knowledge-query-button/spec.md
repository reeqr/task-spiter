## MODIFIED Requirements

### Requirement: 查询考点按钮
系统必须为每个术语提供术语查询按钮组，按钮集合由已启用的查询动作配置动态决定。

#### Scenario: 按钮显示
- **WHEN** 系统渲染术语列表项
- **THEN** 每个术语项必须显示所有已启用的查询动作按钮

#### Scenario: 按钮位置
- **WHEN** 系统布局术语列表项
- **THEN** 查询动作按钮必须位于术语项右侧操作区域

#### Scenario: 按钮样式
- **WHEN** 系统渲染术语查询按钮组
- **THEN** 查询按钮必须使用小尺寸(size="small")且包含图标

#### Scenario: 查询状态隔离
- **WHEN** 用户触发任一查询动作
- **THEN** 系统必须仅锁定对应动作按钮，不得阻塞其他动作按钮的可见性与交互语义
