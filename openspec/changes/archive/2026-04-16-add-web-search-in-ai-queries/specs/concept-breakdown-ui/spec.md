## ADDED Requirements

### Requirement: 概念拆解支持全局联网增强
系统必须在概念拆解请求中遵循全局联网搜索开关，开关开启时优先尝试联网增强路径。

#### Scenario: 全局开关开启时概念拆解尝试联网
- **WHEN** 用户发起概念拆解且全局联网搜索开关开启
- **THEN** 系统必须在请求中优先尝试注入 `tools.web_search`

#### Scenario: 全局开关关闭时概念拆解保持原路径
- **WHEN** 用户发起概念拆解且全局联网搜索开关关闭
- **THEN** 系统必须按原有概念拆解请求路径执行，不注入联网工具参数

### Requirement: 概念拆解联网失败自动降级
系统必须在概念拆解联网失败或 provider 不支持时自动回退到普通拆解路径。

#### Scenario: Provider 不支持时自动降级
- **WHEN** 当前 provider 在能力表或运行时探测中被判定不支持联网工具调用
- **THEN** 系统必须自动回退到普通概念拆解请求并保持结果可返回

#### Scenario: 联网错误时自动降级
- **WHEN** 概念拆解联网请求发生远端错误或参数错误
- **THEN** 系统必须自动回退到普通概念拆解请求并给出可理解提示
