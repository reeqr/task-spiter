## MODIFIED Requirements

### Requirement: 考点模态框展示
系统必须在模态框中展示术语的相关考点，并在联网增强模式下展示可追溯来源信息。

#### Scenario: 点击按钮打开模态框
- **WHEN** 用户点击"查询考点"按钮
- **THEN** 系统必须打开模态框显示该术语的考点列表

#### Scenario: 模态框标题
- **WHEN** 系统打开考点模态框
- **THEN** 模态框标题必须显示当前术语名称

#### Scenario: 模态框内容
- **WHEN** 系统渲染模态框内容
- **THEN** 系统必须使用 KnowledgePointList 组件展示考点

#### Scenario: 联网增强来源展示
- **WHEN** 当前回答由联网搜索增强生成
- **THEN** 模态框中必须展示来源链接与来源时间信息，且来源与回答可关联

#### Scenario: 关闭模态框
- **WHEN** 用户点击关闭按钮或遮罩层
- **THEN** 系统必须关闭模态框

### Requirement: 考点数据来源
系统必须遵循全局联网开关选择考点数据来源，在未启用联网时使用原有路径，在启用联网时优先使用联网增强结果。

#### Scenario: 未启用联网时复用原有问答路径
- **WHEN** 用户未启用联网搜索
- **THEN** 系统必须使用原有问答路径返回考点内容

#### Scenario: 全局开关开启时使用联网增强
- **WHEN** 全局联网搜索开关开启且模型支持联网工具调用
- **THEN** 系统必须通过联网工具增强后返回考点内容

#### Scenario: 联网失败时自动降级
- **WHEN** 联网增强请求失败或当前模型不支持联网工具调用
- **THEN** 系统必须自动回退到原有问答路径并保持查询可用
