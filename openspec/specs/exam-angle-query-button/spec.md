## ADDED Requirements

### Requirement: 出题角度按钮
系统 MUST 为每个术语提供“出题角度”按钮，用于触发考研命题角度查询。

#### Scenario: 按钮显示
- **WHEN** 系统渲染术语列表项
- **THEN** 每个术语项 MUST 显示“出题角度”按钮

#### Scenario: 按钮位置
- **WHEN** 系统布局术语操作区
- **THEN** “出题角度”按钮 MUST 与“解释”“继续拆解”位于同一右侧操作区域

#### Scenario: 按钮查询状态
- **WHEN** 用户点击“出题角度”按钮且请求进行中
- **THEN** 该按钮 MUST 进入加载态并禁止重复触发，其他术语按钮状态不受影响

### Requirement: 出题角度结果展示
系统 MUST 在独立弹窗中展示术语或概念的命题角度分析结果。

#### Scenario: 查询成功后展示
- **WHEN** AI 返回出题角度结果
- **THEN** 系统 MUST 打开弹窗并展示对应术语名称和结果文本

#### Scenario: 查询失败提示
- **WHEN** AI 查询出题角度失败
- **THEN** 系统 MUST 提示失败信息并恢复按钮可点击状态

#### Scenario: 结果隔离
- **WHEN** 用户连续查询不同术语的出题角度
- **THEN** 每次弹窗内容 MUST 对应当前术语，不得复用上一次术语结果
