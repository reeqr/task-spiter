## ADDED Requirements

### Requirement: 提示词面板默认收起
系统 MUST 将提示词模板编辑面板改为默认收起，用户点击后再展开编辑内容。

#### Scenario: 默认进入设置页
- **WHEN** 用户打开模型设置中的提示词模板区域
- **THEN** 模板编辑面板 MUST 默认收起，不展示大面积文本编辑框

#### Scenario: 点击后展开
- **WHEN** 用户点击某个提示词模板项标题
- **THEN** 系统 MUST 展开该模板编辑区域并显示对应输入框

### Requirement: 折叠面板中的模板编辑可用性
系统 MUST 保证折叠交互不影响模板编辑、保存和恢复默认能力。

#### Scenario: 编辑并保存模板
- **WHEN** 用户展开模板项并修改内容后点击保存
- **THEN** 系统 MUST 正常持久化模板内容

#### Scenario: 恢复默认
- **WHEN** 用户点击恢复默认
- **THEN** 所有模板（含新增“出题角度”模板）MUST 恢复为默认值

#### Scenario: 再次打开设置
- **WHEN** 用户关闭并重新打开设置弹窗
- **THEN** 系统 MUST 保持模板内容为最新保存值，折叠状态按默认规则重新初始化
